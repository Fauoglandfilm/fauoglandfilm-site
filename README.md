# Fau&Land Film

Dette er den eneste riktige prosjektmappen videre:

`/Users/tommygarland/Documents/GitHub/fauoglandfilm-site`

Bruk denne mappen i Codex, GitHub Desktop og Vercel.

## Start lokalt

```bash
cd /Users/tommygarland/Documents/GitHub/fauoglandfilm-site
npm install
npm run dev
```

Åpne deretter adressen som vises i terminalen, vanligvis `http://localhost:3000`.

## Bygg prosjektet

```bash
npm run build
```

## Commit og push

```bash
git add .
git commit -m "Din melding"
git push origin main
```

Repoet bruker:

`origin = git@github.com:Fauoglandfilm/fauoglandfilm-site.git`

## Deploy

Vercel deployer automatisk når nye commits pushes til GitHub-repoet.
